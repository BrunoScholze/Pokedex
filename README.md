Pokedex 🕹️


Uma Pokedex criada com Angular e PO UI, integrando a API pública PokeAPI.

🛠 Tecnologias
Angular: Estrutura principal.
PO UI: Componentes para UI.
PokeAPI: Dados dos Pokémons.

🚀 Funcionalidades
Busca por Pokémons 🔍
Filtragem por Tipo 🔖
Detalhes de Pokémons 🐉
Design Responsivo 📱

📥 Instalação
Clone o repositório:
npm install
npm start
